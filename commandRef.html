List of Possible Commands

Most of the ARM Assembly commands have been implemented. The syntax for each available command is listed below. For further information, see the ARM reference site.

Special Commands

These aren't actual ARM functions, but it would be impossible to know what's going on if they didn't exist.


Print binary contents of one register:

PR R0


Print unsigned value of contents of one register:

uVal R0


Print signed value of contents of one register:

sVal R0


Print contents in stack:

PS


Print values of flags:

PF

Move, Add, Subtract

Copy value in one register to another, or binary value of a number (can be negative):

MOV R0, R2
MOV R0, #34


Add two registers or register and a number:

ADDS R0, R0, R0
ADD R0, R0, #42


Add two registers or register and a number with carry:

ADC R0, R0, R0
ADCS R1, R0, #42


Subract last register or numeric value from middle register:

SUB R2, R1, R0
SUBS R2, R1, #42


Subract value in middle register from value in last register:

RSBS R2, R1, R0
RSB R2, R1, #42


Negate a value (converts to two's complement):

NEG R2, R0

*Notes: Always updates flags. Cannot use numeric value in place of second register.

Compare Instructions

Compare:

CMP R0, R2
CMP R0, #42

Bitwise Instructions

'And' one value with another:

AND R2, R1, R0
AND R2, R1, #42


'Or' one value with another:

ORR R2, R1, R0
ORR R2, R1, #42


'Exclusive or' one value with another:

EOR R2, R1, R0
EOR R2, R1, #42


Calculate result of R1 & ~R0 or ~[numeric value]

BIC R2, R1, R0
BIC R2, R1, #42


Calculate result of R1 or-not with the value of register or number specified

ORN R2, R1, R0
ORN R2, R1, #42


Copy ~R0 or ~42 to R2

MVN R2, R0
MVN R2, #42


Bitfield Instructions

Clear width number of bits, starting at lsb

BFC R0, #lsb, #width


Replace width number of bits, starting at lsb, in R1 with bits from R0 (starting at index 0)

BFI R1, R0, #lsb, #width


Multiply/Divide

Standard multiply

MUL R2, R1, R0
MUL R2, R1, #25


R3 = (R2*R1) + R0

MLA R3, R2, R1, R0
