A Simple Example:

LDR R0, =2       ; Load 2 into R0
ADD R1, R0, #4   ; Add 4 to value in R0, store result in Ra
PR R1            ; Print binary contents of R1
uVal R1          ; Print unsigned value of R1


Which would output the following:

00000000000000000000000000000110
6


A Compare Case with Branching:

This example loads 2 into a register, and compares that to 4. If 2 is greater than 4, register R1 will be loaded with the value 7; else, it'll be loaded with the value 42. The result of this operation is printed at the very end.

LDR R0, =2       ; Load 2 into R0
CMP R0, #4       ; Compare 2 to 4
BGE Else         ; If R0 is greater than or equal to 4, go to 'Else'
LDR R1, =42      ; Load 42 into R1 if R0 < 4
B End            ; Go to End
Else:
LDR R1, =7       ; Load 7 into R1 if R0 >= 4
End:
uVal R1          ; Print unsigned value of R1 to console


Which would output the following:

42


A Simple Loop:

This example demonstrates how to make a loop that outputs the binary values between 0 and 10.

LDR R0, =0       ; Load 2 into R0
Top:
PR R0            ; Print current binary contents of R0
CMP R0, #10      ; Compare R0 to 10
BGE End          ; If R0 is greater than or equal to 10, go to 'End'
ADD R0, R0, #1   ; Increment value in R0 by 1
B Top            ; Go to Top
End:


Which would output the following:

00000000000000000000000000000000
00000000000000000000000000000001
00000000000000000000000000000010
00000000000000000000000000000011
00000000000000000000000000000100
00000000000000000000000000000101
00000000000000000000000000000110
00000000000000000000000000000111
00000000000000000000000000001000
00000000000000000000000000001001
00000000000000000000000000001010


A Simple IT Block:

This example demonstrates how to write an IT block that will load 55 into a register if 20 is greater than 42, or load 33 otherwise.

LDR R0, =20
CMP R0, #42
ITE GE
LDRGE R1, =55
LDRLT R1, =33
uVal R1

Which would output the following:

33
